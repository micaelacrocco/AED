Insertar un elemento en una lista (al final, la lista no está ordenada):

    LENGUAJE NATURAL: Esta función inserta un nuevo elemento al final de una lista enlazada.

    PRECONDICIONES: La lista debe ser una lista enlazada válida.

    POSTCONDICIONES: El elemento dado se inserta al final de la lista enlazada.

    PSEUDOCODIGO:

        Función InsertarAlFinal(lista, elemento):
            nuevoNodo ← nuevo Nodo
            nuevoNodo.dato ← elemento
            nuevoNodo.siguiente ← Nulo
            
            Si lista es Nulo:
                lista ← nuevoNodo
            Sino:
                nodoActual ← lista
                Mientras nodoActual.siguiente no sea Nulo:
                    nodoActual ← nodoActual.siguiente
                Fin Mientras
                nodoActual.siguiente ← nuevoNodo
            Fin Si

    PSEUDOCODIGO:

        Función BuscarElemento(lista, clave):
            nodoActual ← lista
            
            Mientras nodoActual no sea Nulo:
                Si nodoActual.dato es igual a clave:
                    retornar verdadero
                Fin Si
                nodoActual ← nodoActual.siguiente
            Fin Mientras
            
            retornar falso
