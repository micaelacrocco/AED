EJERCICIO 1: Parte 1
Dado el siguiente código fuente:

public class IdentifyMyParts {
    public static int x = 7;
    public int y = 3;
}

a) ¿Cuáles son las variables de clase?
b) ¿Cuáles son las variables de instancia?

- a. Variables de clase: x
- b. Variables de instancia: y

EJERCICIO 1: Parte 2
¿Cuál es la salida que produce el siguiente código?

IdentifyMyParts a = new IdentifyMyParts();
IdentifyMyParts b = new IdentifyMyParts();
a.y = 5;
b.y = 6;
a.x = 1;
b.x = 2;
System.out.println("a.y = " + a.y);
System.out.println("b.y = " + b.y);
System.out.println("a.x = " + a.x);
System.out.println("b.x = " + b.x);
System.out.println("IdentifyMyParts.x = " + IdentifyMyParts.x);

- Salida: 
a.y = 5
b.y = 6
a.x = 2
b.x = 2
IdentifyMyParts.x = 2
